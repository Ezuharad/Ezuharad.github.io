Projects

This is an incomplete list of projects I've worked on whether for my degree, personal interest, or my employment. Due to certain licensing constraints, not all projects include links to source repositories.

Talos

Experimental symmetric encryption algorithm based on cellular automata.

Includes both a specification, written in typst, and a reference implementation, written in rust. A second reference implementation is also included which makes use of GPU parallelism for encryption and decryption using PyTorch.

Dotfiles

My personal neovim and Linux dotfiles, stored using GNU stow. Notable tool configurations include:

  • Neovim
  • Wezterm
  • Zsh

ImagePatriot

Novel image immunization technique based on distribution theory. Currently under embargo as part of my Master's Thesis.

Mu AE

System for automatic mutation of image autoencoders. Accepted for publication in COMPSAC 2025.

Dice Bot

Specialized dice bot for playing Fantasy Flight's Starwars RPG.

Implements rolling of unique dice, resolution of success or failure conditions, force point tracking, and custom messages when certain criteria are met during rolls.

AiR-TK

Toolkit combining adversarial machine learning frameworks.

Extends the MAIR framework into a larger number of adversarial training and attack implementations for vision models, all unified under a common interface.

DryBones

Road skeletonization pipeline.

Machine learning pipeline for skeletonizing open street map data. Consists of a U-net architecture trained on Binary Crossentropy and Dice loss.

Lorax

Deforestation surveillance system.

Specialized vision transformer for segmenting deforestation regions of the Amazon rainforest. Trained on Sentinel II satellite imagery.

Gomoku Web Server

Server Application for playing Wuziqi (Gomoku)

Containerized application for multiplayer Go matches. Supports multiple simultaneous games.
